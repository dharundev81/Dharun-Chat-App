# Dharun-Chat-App

  Using Socket.io i create my own chat application
  
  Specs 1.NodeJS
  
  Node Initiate
  
        npm init
      
      
  First specify the dependencies in package.json
  
        "socket.io": "*",
        "express": "*"
      
      
  Install dependencies  
  
        npm install
      
      
  Run the app
  
        node server
